# Love Dedication Website

A minimal, Material-themed React single-page website as a love dedication.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your images:
   - Place background-removed photos in `src/assets/`:
     - `her1.png` - First photo of her (used in collage)
     - `her2.png` - Second photo of her (used in gallery)
     - `her3.png` - Third photo of her (used in gallery)
     - `me.png` - Photo of you (used in collage)

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Image Requirements

All images should be:
- Background-removed (transparent PNG)
- High quality
- Properly sized (recommended: 800-1200px width)

The collage section displays `her1.png` and `me.png` side-by-side.
The gallery section displays all three of her photos in a scattered collage style.
